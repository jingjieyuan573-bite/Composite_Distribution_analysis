# Composite_Distribution_Monte_Carlo_simulation
Python scripts for composite distribution Monte Carlo simulation
### Contents

| File | Description |
|------|--------------|
| `Composite_Monte_Carlo_Simulation.ipynb` | Monte Carlo simulation to evaluate the estimation performance of the proposed composite model. |
| `composite_distribution_data_fit.ipynb` | Empirical fitting of the proposed model to Tesla (TSLA) return data. |
| `composite_skew_normal.ipynb` | Implementation and fitting of the Skew-Normal component. |
| `composite_skew_t.ipynb` | Implementation and fitting of the Skew-t component. |
| `yahoof.ipynb` | Data download and preprocessing from Yahoo Finance. |
